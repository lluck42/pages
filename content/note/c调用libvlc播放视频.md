![播放视频截图](/images/6835614-8ece487a13e23fa3.png)

第一点，编译器 gcc 位数要和代码一致！

第二点，要写一些 gcc 启动参数来指定配置
```
        gcc -I /d/test -L /d/test -llibvlc -o demo2.exe demo2.c && ./demo2.exe

        -L dll文件位置 

        -I  .h文件位置
```
